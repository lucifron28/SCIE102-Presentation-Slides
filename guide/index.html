Quadrat Method

Member 4: Measuring Plant Populations

What is the Quadrat Method?

"This method involves partitioning several randomly distributed sections of habitat with markers, such as string or stakes, and counting the individuals in each quadrat"

Best For

üå± Plants
üêå Very small organisms
ü¶é Slow-moving organisms

Interactive Quadrat Placement

0 organisms found
Density = 0 organisms per quadrat

Procedure Steps

  1. Mark off square areas with stakes and string
  2. Place quadrats randomly in the habitat
  3. Count individuals within each quadrat boundary
  4. Use multiple samples for accuracy
  5. Calculate population estimates

Mark and Recapture Method

Member 4: Measuring Mobile Animals

What is Mark and Recapture?

"First, a random sample of individuals from a habitat are captured, marked (e.g., with tags, paints, or bands) and re-released. At a later date, a second random sample is captured, which includes some of the marked animals from the first sample."

Best For

üê¶ Birds
üêü Fish
ü¶å Mammals

The Lincoln-Petersen Formula

N = (M √ó C) / R
N = Population estimate
M = Number marked in first sample
C = Number caught in second sample
R = Number of marked individuals recaptured

Interactive Calculation Tool

N = (50 √ó 40) / 8
N = 2000 / 8
N = 250 individuals

Important Assumptions

Equal capture probability for marked and unmarked animals
No animals died, were born, or moved between time points
Marks don't affect animal behavior or survival
Marked animals mix back into the population
